# Wallet Module

The **Wallet module** provides functionality for creating and managing Quantova wallets. It includes methods for generating wallets from a mnemonic phrase (HD wallet), managing private keys, encrypting/decrypting keystores, and handling wallet-related tasks for secure interaction with the Quantova blockchain.

## Purpose

The Wallet module ensures secure wallet creation, management, and interaction. It allows users to generate wallets, manage private keys, and encrypt/decrypt sensitive data, enabling secure Quantova transactions.
