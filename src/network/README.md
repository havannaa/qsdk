# Network Module

The **Network module** provides functionality for managing network connections to Quantova nodes. It includes utilities for handling different types of providers (e.g., HTTP, WebSocket), managing network configurations, and ensuring reliable connections to the blockchain.

## Purpose

The Network module ensures that the SDK can reliably connect to Quantova nodes, whether through direct connections or through intermediary proxies. It allows you to manage network configurations and ensure that requests are sent to the appropriate network.
